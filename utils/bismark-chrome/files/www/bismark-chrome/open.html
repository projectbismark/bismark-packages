OpenWrt

OpenWrt is the base project underlying on which BISmark is build. OpenWrt is a Linux distribution that provides a kernel and package repository that allows the customized use of OpenWrt for a variety of applications.

  • License: GPLv2 + packages under various free/open source licenses
  • Source code: Subversion and Git repositories
  • Contributors
